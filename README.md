# 📶 Hotspot Inteligente para Marketing Local com MikroTik Facebook



Esse projeto seria usado em um mikrotik para um hotspot de venda de internet que obrigaria a pessoa a compartilhar no feed o comercio, ao clicar ou sendo redirecionada a pessoa ja é redirecionada para o facebook onde ela é obrigada a postar foto ou texto e ja com a localizacao e pagina comercial marcada, o que gera engajamento


## Sobre o projeto


Este projeto foi desenvolvido como um estudo de integração entre **HTML, JavaScript, Facebook SDK e MikroTik Hotspot**, com o objetivo de transformar o acesso gratuito à internet em uma ferramenta de divulgação para estabelecimentos comerciais.

A proposta consiste em apresentar uma página personalizada ao cliente antes da liberação do acesso ao Wi-Fi. Nessa página, o usuário pode visualizar


---

## Funcionalidades implementadas

* Exibição dinâmica do horário atual utilizando JavaScript.
* Manipulação do conteúdo da página através do DOM.
* Configuração de metadados Open Graph para compartilhamento em redes sociais.
* Integração com o SDK do Facebook.
* Botão de "Curtir" para página do Facebook.
* Botão para compartilhamento direto no feed do Facebook.
* Exemplo de formulário HTML utilizando validação nativa para entrada numérica.

---

## Tecnologias utilizadas

* HTML5
* JavaScript (Vanilla JS)
* Facebook SDK
* Open Graph Protocol

---

## Objetivos de aprendizado

Durante o desenvolvimento deste projeto foram explorados conceitos como:

* Estrutura básica de páginas HTML.
* Manipulação de elementos utilizando JavaScript.
* Atualização dinâmica de conteúdo na página.
* Integração com APIs de terceiros.
* Compartilhamento de conteúdo em redes sociais.
* Utilização de metadados para pré-visualização de links.

## Finalidade

Este projeto faz parte da minha evolução prática na área de desenvolvimento web e representa um laboratório para testes de funcionalidades que posteriormente podem ser aplicadas em sistemas maiores e mais completos.

Cada funcionalidade implementada contribuiu para o aprendizado sobre interação entre front-end e serviços externos, servindo como base para projetos mais avançados envolvendo automação, APIs e desenvolvimento de aplicações web.
